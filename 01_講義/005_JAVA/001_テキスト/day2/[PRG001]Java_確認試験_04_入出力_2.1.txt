■入出力
1.
以下の文はどのような処理を行うか。正しいものを選べ。

BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
br.readLine();

1. コンソール上に文字列を出力する。
2. コンソール上に写真画像を出力する。
3. コンソール上にテキストファイルの内容を出力する。
4. コンソール上で文字を入力できるようにする。

正答：4
解説：
『東京ITスクールJavaテキスト』 第5章/入力と出力/キーボードからの入力 
問題の文はコンソール上で文字列を入力できるようにするための処理です。そのため、選択肢4が正解です。
1行目の文は、文字列を入力できるようにする機能を持った情報を生成している処理です。
2行目の文は、その情報から「readLine()」と呼ばれる処理を実行し、コンソールを入力待ち状態にする処理です。
この時点でユーザがコンソール上で文字列をキーボード入力すると、入力した内容がreadLine()の処理で受け取られます。
なお、コンソール上での入力処理のことを「標準入力」と呼びます。

■入出力
2.
以下の文はどのような処理を行うか。正しいものを選べ。

BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
br.readLine();
br.readLine();
br.readLine();

1. コンソール上に文字列を3回出力する。
2. コンソール上で文字列を3回入力できるようにする。
3. 変数brに値の代入を3回行う
4. 変数brの値の出力を3回行う

正答：2
解説：
『東京ITスクールJavaテキスト』 第5章/入力と出力/複数の文字列の入力 
問題の文はコンソール上で文字列を3回入力できるようにするための処理です。そのため、選択肢2が正解です。
