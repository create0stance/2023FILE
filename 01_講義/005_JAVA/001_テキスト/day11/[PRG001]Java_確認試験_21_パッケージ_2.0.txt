■パッケージ
1.
パッケージの用途として正しいものはどれか。

1. 同名のクラスを区別するため。
2. 特定のクラスを他のクラスからアクセスできないようにするため。
3. 特定のクラスのオブジェクトを抽象クラスとして使用できるようにするため。
4. 複数のクラス間で継承関係を持たせるため。

正答：1
解説：
『東京ITスクールJavaテキスト』 第24章 パッケージ/パッケージとは
システムの規模が大きくなると、クラスの数が増え、便宜上同じ名前のクラスを定義しなければならない場合があります。
その際に同名のクラスを区別するための情報としてパッケージが利用されます。
たとえ同じ名前のクラスが複数あったとしても、それぞれが異なるパッケージ内に入っている場合、それらのクラスはシステム上は別物のクラスとして扱われます。
したがって、選択肢1が正解です。


■パッケージ
2.
クラス内にインポート文を記述する利点として正しいものはどれか。

1. 同じパッケージ内のクラスから、自動的にオブジェクトを生成できる。
2. 異なるパッケージ内のクラスとの間に継承関係を持たせられる。
3. 同じパッケージ内のクラスに、パッケージ名のみをメソッド内に記述してアクセスできる。
4. メソッド内にクラス名のみを記述して、異なるパッケージ内のクラスにアクセスできる。

正答：4
解説：
『東京ITスクールJavaテキスト』 第24章 インポート/インポートとは
他のパッケージ内のクラスをメソッド内で利用する場合、そのクラスを利用する度に「パッケージ名.クラス名」と記述しなければなりません。
しかし、この記述方法ではソースコードが冗長的に長くなってしまいます。
そこで利用されるのがインポート分です。ソースコード上部で予め「パッケージ名.クラス名」と記述することで、それ以降でクラスを利用する際にパッケージの記述を省略できます。
したがって、選択肢4が正解です。
