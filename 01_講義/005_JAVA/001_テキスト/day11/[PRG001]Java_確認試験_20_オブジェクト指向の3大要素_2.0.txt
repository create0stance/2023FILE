■オブジェクト指向の3大要素
1.
オブジェクト指向の3大要素として誤っているものはどれか。

1. カプセル化
2. 再利用
3. 継承
4. ポリモーフィズム(多態性)

正答：2
解説：
『東京ITスクールJavaテキスト』 第23章 3大要素とは
オブジェクト指向の3大要素は、カプセル化、継承、ポリモーフィズム(多態性)の3つです。
したがって正解は2となります。

■オブジェクト指向の3大要素
2.
オブジェクト指向の3大要素の説明として正しいものはどれか。

1. カプセル化を用いることで保守性の向上やバグに強いプログラムの作成に繋がる
2. 継承を用いることで開発者が把握すべき情報が最小限に抑えられる
3. ポリモーフィズムを用いることでプログラムの再利用性を高め、保守・修正を容易にすることができる
4. 再利用を用いることで開発工数を最小限に抑えることができる

正答：1
解説：
『東京ITスクールJavaテキスト』 第23章 3大要素とは
2はポリモーフィズムの説明、3は継承の説明となります。
4の再利用はオブジェクト指向の3大要素ではありません。
したがって、選択肢1が正解です。
