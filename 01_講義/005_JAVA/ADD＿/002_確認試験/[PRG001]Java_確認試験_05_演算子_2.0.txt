■演算子
1. 「%」という演算子はどのような処理を行うか。正しいものを選べ。

1. 2つの数の割合（パーセント計算）を計算する。
2. 2つの数の割り算の余りを計算する。
3. 2つの数の割り算の商を計算する。
4. 2つの数の掛け算を計算する。

正答：2
解説：
『東京ITスクールJavaテキスト』 第6章/演算子/演算子の種類
「%」は剰余（割り算の余り）を計算するための演算子です。そのため、選択肢2が正解です。
例えば、「int x = 10 % 3」という式があった場合、計算結果は1であり、その値はxに代入されます。


■演算子
2.
以下の文はどのような処理を行うか。正しいものを選べ。
ただし、2つの変数num1とnum2には予め整数値0が代入されているとする。

num2 = num1++;

1. num1の値がnum2に代入された後、num1の値が1加算される。
2. num1の値がnum2に代入された後、num1とnum2それぞれの値が1加算される。
3. num1の値が1加算された後、 num1の値がnum2に代入される。
4. num1とnum2それぞれの値が1加算された後、num1の値がnum2に代入される。

正答：1

解説：
『東京ITスクールJavaテキスト』 第6章/演算子の種類/インクリメントとデクリメント
++はインクリメントという演算子で、「もとからある数字に1を足す」という計算を行います。
インクリメント演算子には後置き（a++)と前置き(++a)の2種類があり、それにより計算を実施するタイミングが変わります。
後置きは他の演算子の処理がすべて終わった際に実施され、前置きは他の演算子の処理の前に実施されます。
そのため、問題の文の処理は選択肢1のようになります。


■演算子
3.
以下の式はどのような処理を行うか。正しいものを選べ。

"答え：" + 2 + 10;

1. 数値2と数値10の足し算が行われる。次に、その数値と文字列「答え：」が文字列連結する。
2. 文字列「答え：」と数値2が文字列連結する。次に、数値2と数値10の足し算が行われる。
3. 文字列「答え：」と数値2が文字列連結する。次に、その文字列と数値10が文字列連結する。
4. 型が異なる値で計算しようとするため、エラーになる。

正答：3
解説：
『東京ITスクールJavaテキスト』 第6章/演算子の種類/文字列連結演算子
「+」の演算子は2つのオペランドを持つことで、2種類の演算子のいずれかの機能を持ちます。
2つのオペランドが両方とも数値の場合は加算のための演算子になります。また、2つのオペランドの内、型法でも文字列だった場合は文字列連結のための演算子となります。
また、「+」は左結合の特性を持っており、「+」が1つの式で複数回記述されている場合、左側に書かれた演算子から処理されます。
そのため、問題の文では、まず「答え：」という文字列と数値2の間で文字列連結が行われ、次に「答え：2」という文字列と数値10の文字列連結が行われます。
そして、この式の結果は「応え：210」となります。
そのため、選択肢3が正解です。


■演算子
4.
型変換に関する説明として誤っているものはどれか。

1. 小さい型の値を大きい型の変数に代入すると、その値は変数と同じ型に変換される。
2. キャスト演算子を使用することで、大きい型の値を小さい型の変数に代入できる。
3. double型の値をint型に変換した場合、小数点以下の情報は失われる。
4. int型の値とdouble型の値で演算を行った場合、double型の値はint型に変換される。

正答：4
解説：
『東京ITスクールJavaテキスト』 第6章/型変換/サイズが小さい型に代入する
小さい型の値を大きい型の変数に代入すると、暗黙的な型変換が実行され、値は変数の型に合わせて変換されます。そのため、選択肢1は不正解です。
また、double型はint型よりも大きいため、double型の値をint型の変数に直接代入することはできません。
しかし、キャスト演算子を使用して明示的にdouble型をint型に変換すれば、int型への代入も可能となります。
このとき、double型はint型に変換された際に小数点以下の情報が切り捨てられます。そのため、選択肢2と選択肢3は不正解です。
また、型の異なる値同士で演算を行うことは可能です。その場合は、小さい型の値が大きい型に合わせて暗黙的に変換されます。
例えば、double型の値とint型の値で演算した場合は、int型の値がdouble型に変換されます。そのため、選択肢4が正解です。
