■概要
1.
Javaのクラスファイルが持つメリットとして正しいものはどれか。

1. Java以外の言語を実行させるためのソフトウェアでもJavaのプログラムを実行できる。
2. OS環境に依存せずプログラムを実行できる。
3. Windows環境でのみプログラムを実行できる。
4. Windows環境以外でのみプログラムを実行できる。

正答：2
解説：
『東京ITスクールJavaテキスト』第3章/プログラムのコンパイルと実行/Javaのコンパイルの仕組み
大半のプログラミング言語では、プログラムの開発環境と実行環境が一致していないと実行できません。
それに対して、Javaのコンパイラによって作成されたクラスファイルは、Javaが実行可能であればどの環境でも実行可能です。環境に依存しない点はJavaの大きな強みでもあります。
以上のことから、選択肢2が正解です。


■概要
2.
下記の文中の（　）内にあてはまる単語の組み合わせとして、正しいものはどれか。

Javaでは、（ ① ）というソフトウェアを使用して、ソースファイルからクラスファイルを作成する。そして、（ ② ）というソフトウェアを使用して、クラスファイルの内容を機械語に変換し、プログラムを実行する。

1. ①：ソースコード、②：バイトコード
2. ①：バイトコード、②：ソースコード
3. ①：コンパイラ、②：インタプリタ（Java仮想マシン）
4. ①：インタプリタ（Java仮想マシン）、②：コンパイラ

正答：3
解説：
『東京ITスクールJavaテキスト』第3章/プログラムのコンパイルと実行/Javaのコンパイルの仕組み
大半のプログラミング言語ではコンパイラ、もしくはインタプリタのいずれかを使用してプログラムを実行します。
それに対して、Javaではコンパイラとインタプリタによる2段階での処理を通してプログラムが実行されます。
なお、Javaのインタプリタのことを「Java仮想マシン（Java Virtual Machine、またはJVM）」と呼びます。
以上のことから、選択肢3が正解です。
