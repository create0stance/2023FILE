
■例外
1.
catchブロック内の文が実行される条件として正しいものはどれか。

1. tryブロック内で送出された例外のクラスが、配列に関連するものだった場合に実行される。
2. tryブロック内で何らかの例外が送出された場合に実行される（クラスは問わない）。
3. tryブロック内で送出された例外のクラスが、catchブロックの引数で指定されたものに該当する場合に実行される。
4. tryブロック内で例外が送出されずに処理が完了した場合に実行される。

正答：3
解説：
『東京ITスクールJavaテキスト』 第25章 例外処理/try-catch-finally文
tryブロック内の処理で何らかの例外が送出され、かつその例外のクラスがcatchブロックの引数の型として指定されたクラスと合致する場合、catchブロック内の文が実行されます。
したがって、選択肢3が正解です。
なお、catchブロックの引数として指定されたクラスが送出された例外クラスのスーパークラスである場合も、合致すると判定されます。


■例外
2.
finallyブロック内に記述された文の特徴として正しいものはどれか。

1. tryブロック内で例外が送出されなかった場合のみ実行される。
2. tryブロック内で何らかの例外が送出された場合のみ実行される。
3. tryブロック内で何らかの例外が2回以上送出された場合のみ実行される。
4. tryブロック内で例外が送出されたかどうかに関わらず実行される。

正答：4
解説：
『東京ITスクールJavaテキスト』 第25章 例外処理/try-catch-finally文
finallyブロック内に記述された文は、tryブロック内で例外が送出された、されないに関わらず必ず実行されます。
そのため、finallyブロックは最後に必ず実行してほしい処理がある場合に最適です。したがって、選択肢4が正解です。


■例外
3.
例外処理の記述が不要となるのは、送出された例外がどのようなクラスの場合か。正しいものを選べ。

1. Throwableクラスのサブクラス、またはRuntimeExceptionクラスのサブクラスの場合。
2. Exceptionクラスのサブクラス、またはRuntimeExceptionクラスのサブクラスの場合。
3. Errorクラスのサブクラス、またはRuntimeExceptionクラスのサブクラスの場合。
4. クラス名の末尾が「Exception」と記述されていないクラスの場合。

正答：3
解説：
『東京ITスクールJavaテキスト』 第25章 例外を投げる/throws文 
例外処理は必ず記述しないといけないわけではありません。
「送出された例外がErrorクラスのサブクラスであるか」、「送出された例外がRuntimeExceptionクラスのサブクラスであるか」のいずれかを満たす場合、例外処理の記述は不要となります。
したがって、選択肢3が正解です。
