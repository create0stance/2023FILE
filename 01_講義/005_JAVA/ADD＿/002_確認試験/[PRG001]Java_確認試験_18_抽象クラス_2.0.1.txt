■抽象クラス
1.
抽象クラスの特徴として正しいものはどれか。

1. 抽象クラスからオブジェクトを生成できる。
2. 抽象クラスは抽象メソッドのみを持っている。
3. 抽象クラスには抽象メソッドを定義できる。
4. 抽象クラスと通常のクラスは文法上の違いはない。

正答：3
解説：
『東京ITスクールJavaテキスト』 第21章
抽象クラス/抽象クラスでは抽象メソッドを定義できます。
したがって、正解は3となります。
なお、抽象クラス内に具象メソッドも定義できるため、2は不正解となります。


■抽象クラス
2.
次の（　）内にあてはまる単語はどれか。
抽象クラス、抽象メソッドは（　）修飾子をつけて定義する。

1. private
2. public
3. abstract
4. extends

正答：3
解説：
『東京ITスクールJavaテキスト』 第21章 抽象クラス/抽象クラスと抽象メソッドの定義
抽象クラス、抽象メソッドはabstract修飾子をつけて定義します。
したがって、選択肢3が正解です。
