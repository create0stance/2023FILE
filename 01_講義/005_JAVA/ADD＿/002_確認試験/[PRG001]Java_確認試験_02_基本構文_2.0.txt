■基本構文
1.
下記の文はどのような処理を行うためのものか。正しいものを選べ。

System.out.println()

1. コンソール中で文字を入力できるようにする。
2. コンソール上に文字列を出力する。
3. コンソール上に写真画像を表示する。
4. コンソール上を自動的に閉じる。

正答：2
解説：
『東京ITスクールJavaテキスト』第3章/画面への出力/画面に出力する
「System.out.println()」は文字列をコンソール上に出力する処理です。このような出力処理を「標準出力」と呼びます。
なお、上記の文による出力処理では、出力内容の末尾では必ず改行が挿入されます。
以上のことから、選択肢2が正解です。


■基本構文
2.
main()メソッドに関する説明として正しいものはどれか。

1. プログラム起動時にmain()メソッド内に記述した文が最初に実行される。
2. main()メソッドのブロックは、クラスブロックの外側に記述しなければならない。
3. main()メソッドのブロックは、" [ " と " ] " で囲むかたちで表記する。
4. main()メソッドの記述は省略可能であり、実行したい文をクラスブロック内に直接記述することもできる。

正答：1
解説：
『東京ITスクールJavaテキスト』第3章/コードの内容/main()メソッド
「public static void main(String[] args)」から始まるブロックは「main()メソッド」と呼ばれ、プログラム実行時にこのブロック内の処理が一番最初に実行されます。
以上のことから、選択肢1が正解です。
なお、main()メソッドのブロック手前の記述はJavaで定められた決まった書き方です。自由に書き換えてしまうと、main()メソッドとして機能しなくなるため、注意してください。


■基本構文
3.
エスケープシーケンスに関する説明として正しいものはどれか。

1. $マークとアルファベット1文字を組み合わせて、1つの文字リテラルとして扱われる。
2. 改行や水平タブを特殊文字として文字列中に記述できる。
3. 文字列を出力する際には、エスケープシーケンスも必ず記述しなければならない。
4. エスケープシーケンスの表記に使用する記号は、文字として出力することができない。

正答：2
解説：
『東京ITスクールJavaテキスト』第3章/文字と数値/エスケープシーケンス
エスケープシーケンスとは、\マークと特定の文字を組み合わせた記述のことであり、文字として表現できないものを疑似的に表現できるようにしたものです。文字列リテラルのように見えますが、扱いとしては文字リテラルになります。
エスケープシーケンスの中でも特に代表的なものに、「\n」と「\t」があります。
「\n」は改行1個分、「\t」は水平タブ（Tab空白）1個分を挿入したことと同じ効果を持ちます。
以上のことから、選択肢2が正解です。
