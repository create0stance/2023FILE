力試し◆電卓編

下記の条件に合った四則演算ができるプログラムを作成してください。

〇条件
・抽象メソッドを持つクラスClacを使用し、抽象メソッドcalculationを定義すること。
・足し算、引き算、掛け算、割り算のクラスはそれぞれ別のクラス。
・足し算、引き算、掛け算、割り算クラスのフィールドはprivate修飾子であること。
・足し算、引き算、掛け算、割り算クラスはClacを継承していること。

・mainメソッド内に書かれている記述は以下の通り。
    //足し算を行う。
    Plus pl1 = new Plus(3,2);
    Plus pl2 = new Plus();
    pl2.calculation(10,2)

     //引き算を行う。
     Minus mi1 = new Minus(3,2);
     Minus mi2 = new Minus();
     mi2.calculation(10,2)

     //かけ算を行う。
     Multiplication mu1 = new Multiplication(3,2);
     Multiplication mu2 = new Multiplication();
     mu2.calculation(10,2)

      //わり算を行う。
      Division di1 = new Division(3,2);
      Division di2 = new Division();
      di2.calculation(10,2)

実行した結果、コンソールには下記の通り表示された。

3 + 2 = 5.0
10 + 2 = 12.0
3 - 2 = 1.0
10 - 2 = 8.0
3 * 2 = 6.0
10 * 2 = 20.0
3 / 2 = 1.5
10 / 2 = 5.0

①上記の実行結果となるように実装しなさい。

②Clacクラスに checkNumメソッドを実装し、
　calculationの引数に負の数が渡された場合はコンソールに下記の通り表示されるようにしなさい。

　負の数は計算できません。

③標準入力を使い、ユーザーが足し算、引き算、掛け算、割り算を選び、
　そのあと入力された値を計算するように修正してください。
