力試し◆RPGのあるある編

Lv1.
　コンソールに下記のような文言を表示し、
　ユーザーの入力が、はい、いいえで表示が変わるプログラムを作成してください。
　※出力された文言は、おおよそあっていれば細部は何でも良い。

    （はいの場合）
    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    はい
    「やった！これからよろしくね」

    （いいえの場合）
    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    いいえ
    「そんな…。一緒に連れて行ってよ…」

Lv2.
Lv1で作成したプログラムで、いいえが入力された場合、
はいが入力されるまでループされるように修正してください。

    （はいの場合）
    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    はい
    「やった！これからよろしくね」

    （いいえの場合）
    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    いいえ

    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    いいえ

    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    はい
    「やった！これからよろしくね」

Lv3. 
いいえを選んでループを２回以上した場合、
はいを選んだときに、追加のセリフが表示し、断った回数が表示されるように修正してください。

    （いいえの場合）
    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    いいえ

    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    いいえ

    「僕を旅の仲間に入れてください！」
    仲間に入れますか？
    はい
    「やった！これからよろしくね」
    「２回も断られて、内心ヒヤヒヤしたよ…」
