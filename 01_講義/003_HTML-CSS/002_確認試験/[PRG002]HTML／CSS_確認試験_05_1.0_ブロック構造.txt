■ブロック構造
1.
ブロック要素とインライン要素に関する説明として正しいものはどれか。

1. ブロック要素には、<p>タグや<h1>タグなど、HTML文書の構造を決めるタグが分類される。
2. インライン要素には、<p>タグや<h1>タグなど、HTML文書の構造を決めるタグが分類される。
3. グルーピングするためブロック要素として、<span>タグが挙げられる。
4. 文字の装飾を行うためのインライン要素として、<div>タグが挙げられる。

正答：1
解説：
『HTML/CSS』ブロック構造
ブロック要素には、HTML文書の構造を決めるタグが分類されます。また、インライン要素には、テキスト、画像関連、リンクのタグが分類されます。したがって、選択肢1が正解です。


■ブロック構造
2.
paddingプロパティの機能について正しいものはどれか。

1. ブロックの横幅を変更する。
2. ブロックの外側の余白を空ける。
3. ブロックの内側の余白を空ける。
4. ブロックに枠線を表示する。

正答：3
解説：
『HTML/CSS』ブロック構造/コンテンツの大きさと余白、枠線を制御する/ブロック要素の余白(paddingプロパティ/marginプロパティ)
paddingプロパティはブロックの内側の余白を空けるためのプロパティです。したがって、選択肢3が正解です。
