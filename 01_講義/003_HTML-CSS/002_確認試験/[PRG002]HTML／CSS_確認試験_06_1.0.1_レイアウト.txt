■ページレイアウト
1.
要素を横並びにするプロパティはどれか。

1. rightプロパティ
2. floatプロパティ
3. leftプロパティ
4. sideプロパティ

正答：2
解説：
『HTML/CSS』ページレイアウト/floatプロパティによる段組み
要素を横並びにする時、floatプロパティを使用します。floatプロパティを指定しなければ、要素が縦に並びます。
したがって、選択肢2が正解です。


■ページレイアウト
2.
floatの解除する方法に関する説明として誤っているのはどれか。
なお、floatの効果が適用されているブロックは左詰めで横並びになっているものとする。

1. CSSに「clear:both;」を追加する。
2. CSSに「float:delete;」を追加する。
3. CSSに「overflow:hidden;」を追加する。
4. CSSに「clear:left;」プロパティを追加する。

正答：2
解説：
『HTML/CSS』ページレイアウト/floatの解除
「float」プロパティに付与できる値に「delete」というものは存在しません。また、「float」プロパティではfloatの解除はできません。
問題文のケースでは、floatを解除するには「clear:both;」、「overflow:hidden;」、「clear:left;」のいずれかを使用します。
したがって、選択肢2が正解です。
