■プロパティ
1.
文章の表示位置を指定するプロパティはどれか。

1. text-decorationプロパティ
2. font-weightプロパティ
3. text-alignプロパティ
4. colorプロパティ

正答：3
解説：
『HTML/CSS』プロパティの種類/文字装飾/文字の表示位置を指定する。（text-align）
表示位置を指定するのは、text-alignプロパティです。したがって、選択肢3が正解です。
なお、text-decorationプロパティは下線を引く・font-weightプロパティは太さの指定・colorプロパティは色の指定となります。

■プロパティ
2.
background-imageの正しい説明はどれか。


1. 背景画像の表示サイズを決める
2. 背景色を決める
3. 文章と文章の間に画像を埋め込む
4. 背景に画像を配置する

正答：4
解説：
『HTML/CSS』プロパティの種類/背景装飾/backgroundプロパティのバリエーション
background-imageはCSSのプロパティで、要素に1つ以上の背景画像を設定します。したがって、選択肢4が正解です。
