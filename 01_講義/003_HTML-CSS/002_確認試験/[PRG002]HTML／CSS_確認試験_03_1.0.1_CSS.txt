■CSS
1.
CSSに関する説明として正しいものはどれか。

1. 波カッコ({})のブロックの外側にプロパティ、内側にセレクタを記述する。
2. 要素内のstyle属性の値には、「プロパティ:値;」という形式で記述する。
3. CSSのコードは、HTMLのコード内にのみ記述できる。
4. タイプセレクタにより、タグのclass属性の値で見た目の変更箇所を指定できる。

正答：2
解説：
『HTML/CSS』CSSとは/CSSの基本書式
要素内に記述されたstyle属性の値には「プロパティ名:値;」という形式で記述します。したがって、選択肢2が正解です。


■CSS
2.
下記のようなCSSのセレクタを記述した場合、HTML中に記述された情報でセレクタの参照対象となるのはどれか。

#test

1. class属性の値
2. id属性の値
3. 要素名
4. プロパティ

正答：2
解説：
『HTML/CSS』CSSとは/セレクタの指定方法/[3] IDセレクタ
「#」から始まるセレクタは「IDセレクタ」と呼ばれ、該当するID属性が記述された要素がセレクタの対象となります。したがって、選択肢2が正解です。
