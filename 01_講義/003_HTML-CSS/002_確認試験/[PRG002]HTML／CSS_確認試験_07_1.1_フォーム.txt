■フォーム
1.
<form>タグに関する説明として誤っているものはどれか。

1. method属性の値に"get"と指定すると、入力した値がURL上から表示される。
2. <form>タグにname属性を記述することで、入力した複数の値を区別することができる。
3. 一般的に、<input>タグは<form>タグの子要素として記述する。
4. action属性には入力した値の送信先URLを記述する。

正答：2
解説：
『HTML/CSS』フォーム/フォーム領域
name属性は、<input>タグ、<textarea>タグ、<select>タグなどのコントロールに記述する属性です。
この属性を記述することで、各コントロールの入力値を区別することができます。
なお、<form>タグ自体を区別するためにname属性を用いることもありますが、入力した複数の値を区別するためには
inputタグに対してname属性を設定します。したがって、選択肢2が正解です。

■フォーム
2.
<input>タグ、<option>タグ内に記述できるvalue属性の役割として正しいものはどれか。

1. コントロールの横幅を指定する。
2. 入力値(初期値)を指定する。
3. 各コントロールを識別するための名前を指定する。
4. 入力値の送信方式を指定する。

正答：2
解説：
『HTML/CSS』フォーム/コントロール/<input>タグ
value属性は各コントロールの入力値を指定するための属性です。したがって、選択肢2が正解です。
なお、value属性に値を指定することで、その値が対象のコントロールの入力値（初期値）として扱われます。
