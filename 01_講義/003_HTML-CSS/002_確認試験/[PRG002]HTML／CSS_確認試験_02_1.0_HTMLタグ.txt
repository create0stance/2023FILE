■HTMLタグ
1.
<p>タグの機能として正しいものはどれか。

1. 文の間に改行を挿入する。
2. 文章を段落分けする。
3. 文章中に見出しを表示する。
4. 箇条書きを表示する。

正答：2
解説：
『HTML/CSS』HTMLタグの種類/段落
<p>タグは、文章を段落分けして出力する機能を持ちます。したがって、選択肢2が正解です。


■HTMLタグ
2.
見出しを表示するためのタグはどれか。

1. <br>タグ
2. <h1>タグ
3. <p>タグ
4. <ul>タグ

正答：2
解説：
『HTML/CSS』HTMLタグの種類/見出し
見出しを表示するためのタグは<h1>タグ〜<h6>タグまで用意されています。したがって、選択肢2が正解です。


■HTMLタグ
3.
以下の2つの<img>タグに関する説明として正しいものはどれか。

<img src="../images/sample1.jpg" />
<img src="http://www.sss.co.jp/images/sample2.jpg" />

1. どちらの画像も相対パスで指定されている。
2. どちらの画像も絶対パスで指定されている。
3. "sample1.jpg"は相対パスで、"sample2.jpg"は絶対パスで指定されている。
4. "sample1.jpg"は絶対パスで、"sample2.jpg"は相対パスで指定されている。

正答：3
解説：
『HTML/CSS』HTMLタグの種類/リンク/相対パスと絶対パス
相対パスは、現在出力されているファイルからの相対的な位置で遷移先を指定する記述形式です。遷移先の指定の際には、「ファイル名」、「フォルダ名」、「.」、「..」などから記述します。
また、絶対パスは「http://」や「https://」から始まるURLを指定する記述形式です。
したがって、選択肢3が正解です。
