■HTML概要
1.
HTMLに関する説明として誤っているものはどれか。

1. HTMLはWebブラウザ上で動作する。
2. HTMLはバージョンの違いによる機能の差異がない。
3. HTMLの各要素には属性を指定できる。
4. HTMLの各要素は入れ子構造で記述できる。

正答：2
解説：
『HTML/CSS』HTML/文章型宣言
HTMLはバージョンに記述できるタグや文法上の制約に違いが出てきます。したがって、選択肢2が正解です。
なお、HTMLのバージョン指定は文書型宣言で行います。


■HTML概要
2.
<meta>タグに関する説明として正しいものはどれか。

1. Webブラウザ上部にページタイトルを表示できる。
2. HTMLのバージョンを指定できる。
3. 子要素として<body>タグを記述する。
4. HTML文書の文字コードを指定できる。

正答：4
解説：
『HTML/CSS』HTML/文字コード
<meta>タグでは、記述された文書自体の形式や文字コードを指定できます。したがって、選択肢4が正解です。

■HTML概要
3.
文書型宣言の役割として正しいものはどれか。

1. HTMLを読み込む際の文字コードを指定する。
2. 表示されたページの背景を指定する。
3. ページタイトルを指定する。
4. 記述されたHTMLのバージョンを指定する。

正答：4
解説：
『HTML/CSS』HTML/文章型宣言
文書型宣言（DOCTYPE宣言）はHTMLのバージョンを指定するために記述します。したがって、選択肢4が正解です。
